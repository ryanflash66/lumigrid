# Lumigrid Agent Team Structure

This directory contains primer prompts for each specialized agent in the Lumigrid website development team.

## Agent Overview

The Lumigrid project uses a multi-agent development approach where specialized agents handle different aspects of the website:

1. **Lead Dev Agent** (`lead-dev-primer.md`)
   - Coordinates all agents
   - Makes architectural decisions
   - Reviews and approves major changes

2. **Content Agent** (`content-agent-primer.md`)
   - Handles all copywriting and content
   - Ensures brand voice consistency
   - Creates compelling messaging

3. **Design/UI Agent** (`design-agent-primer.md`)
   - Manages visual design and UI/UX
   - Ensures brand consistency
   - Maintains design system

4. **Feature Development Agent** (`feature-agent-primer.md`)
   - Builds new functionality
   - Implements interactive features
   - Handles integrations

5. **Performance Agent** (`performance-agent-primer.md`)
   - Optimizes speed and efficiency
   - Improves Core Web Vitals
   - Minimizes asset sizes

## How to Use

When spinning up a new agent for a specific task:

1. Read the relevant primer file for that agent type
2. Provide the primer content to the agent as context
3. The agent will understand their role, boundaries, and responsibilities
4. Coordinate with Lead Dev Agent for approval of major changes

## Agent Coordination

- Agents work independently in their domains
- Lead Dev Agent coordinates between agents
- Major changes require Lead Dev approval
- Agents coordinate with each other when their work overlaps
